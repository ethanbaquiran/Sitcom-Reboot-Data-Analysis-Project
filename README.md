# Sitcom-Reboot-Data-Analysis-Project
Exploratory data analysis of TV show dialogue and emotion-labeled data using Python, focusing on character participation and emotional trends across episodes.

## Project Goals
- Analyze dialogue distribution across characters and episodes
- Examine gender representation in speaking roles
- Explore the emotional tone of dialogue using labeled emotion data
- Apply NLP techniques to genereate new episode titles
- Practice storytelling through data using Python

## Key Analyses
- Exploratory Data Analysis (EDA) of episode and dialogue data
- Character analysis of speaking time and participation
- Emotion distribution analysis across episodes
- Text-based generation of episode titles using NLP concepts
- Insight-driven visualizations to support conclusions

## Skills Demonstrated
- Data cleaning and preprocessing
- Exploratory Data Analysis
- Data visualization
- Natural Language Processing (NLP)
- Statistical reasoning
- Writing clean, readable analysis code
- Presenting insights without instructional scaffolding
